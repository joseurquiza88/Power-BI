# Power BI – Proyectos de Análisis y Business Intelligence

Este repositorio reúne distintos proyectos desarrollados en **Power BI** como parte de mi proceso de aprendizaje en análisis de datos y **Business Intelligence**.

El objetivo es construir dashboards y modelos de datos que permitan explorar información de forma visual y responder preguntas de negocio. Los proyectos irán evolucionando progresivamente, comenzando con análisis más simples y avanzando hacia desarrollos más completos.

A lo largo de los proyectos se trabaja sobre distintos aspectos del proceso de BI, incluyendo:

* **Extracción de datos** desde diferentes fuentes
* **Transformación y limpieza de datos** mediante **Power Query**
* **Modelado y vinculación de tablas**
* Construcción de **medidas y métricas** para análisis de datos
* Desarrollo de **dashboards interactivos** en Power BI

La idea es ir incorporando gradualmente nuevas herramientas, fuentes de datos y técnicas de análisis, documentando el proceso de aprendizaje desde proyectos iniciales hasta desarrollos más avanzados.

## Contenido

Los proyectos incluidos en este repositorio exploran distintos enfoques de análisis de datos, incluyendo:

* análisis de ventas
* exploración de métricas comerciales
* visualización de datos para toma de decisiones

Cada proyecto incluye el archivo `.pbix` y capturas del dashboard para facilitar su visualización.
